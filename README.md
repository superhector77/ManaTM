To run the .py files on the RasPi, need to set up a virtual environment. Here are the steps: 

In bash, enter:

  
	python3 -m venv ~/servo_env
	
  
	source ~/servo_env/bin/activate

	
Navigate to ManaTMPy using the command:

 	cd ManaTMPy
	
Run the files using commands: 

  
	python3 rudder_motor_pump.py
	
  
	python 3 leak_control.py


for access to the Lidar Github look here:
